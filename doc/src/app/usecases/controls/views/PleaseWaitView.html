Please Wait

This control is a simple blocking control that can be used for blocking the UI while something is loading. It displays an animated block of squares in the middle of the screen.

Demo

Global

To show the loader, press the button. The loader will automatically turn off after 5 seconds.

Local

To show the loader, press the button. The loader will automatically turn off after 5 seconds.

This area will be blocked for 5 seconds.

Setup

Import the control it into your module as follows:

In your HTML then place this where you want the blocker to live:

Global

It all depends where you place the control. If you just put it in an outer tag somewhere and don't control it with style="position: relative" it will by default block as much as it possibly can.

Local

First, put the HTML code at the bottom of the root tag where it should block. Then make sure the parent element is set to style="position: relative" it will by default block as much as it possibly can.
