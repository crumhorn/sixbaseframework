Split Layout View

The split layout control is a further extended version of the "Angular Split" control to cover needs that the original doesn't have. This original control can be found here.

It allows you to have any number of nested, resizable sections on a page, that can also be minimized and maximized (full screen overlay).:

Demo

Here is a rather crude example of how the split layout works. The main view is split horizontally in 2, and the right side of that is split vertically in 2.

Each section is also using the SplitLayoutContainer as a content holder, which has support for header, body, toolbar, and footer.
