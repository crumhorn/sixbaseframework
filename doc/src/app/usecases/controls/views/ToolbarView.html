Toolbar

The Toolbar control mimics how fat client toolbars look and function. It comes with many different button and input types and hooks together with the AgGrid context menu if used together with the ContextMenuHelper, where you only need to enable/disable toolbar buttons for the same menu actions to be enabled/disabled in the context menu at the same time.

If the toolbar runs out of horizontal space, it will either show a scrollbar (if you've set toolbarSettings.enablePopupOverflow = false), or a popup icon on the far right will appear (if you've set toolbarSettings.enablePopupOverflow = true) showing the items that do not fit in the toolbar.

Real Images or Web Fonts

As shown at the top, toolbarSettings.treatImagesAsWebFonts = true; tells the toolbar that every image passed to it via parameters will be treated as a web font. If instead you set it to false, you define the base path to where your toolbar images are via toolbarSettings.imagePath = "assets/images/toolbar/"; and then pass in images in the parameters relative to that path, for example someMethod(..., ..., "image_enabled.png", "image_disabled.png)
