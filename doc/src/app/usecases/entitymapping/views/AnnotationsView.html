Annotations and Decorators

Annotations on properties in your entities defines how the raw json properties are converted into real objects. For the most general usecases of mapping data (for example strings into Date objects), many annotations are provided by the framework.
Example of annotation usage where a string UNIX date would be converted into a JS Date Object
