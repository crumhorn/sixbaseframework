ColumnHelper

This AgGrid related helper, assists with the creation of columns. If you are using the InlineSearch feature, it will also help wrap your cell renderers with the correct code to display inline search.

The goal of the ColumnHelper is to facilitate the setup of AgGrid in a streamlined manner. It's easier to start with a "Demo". The example code is here a full setup of a rather complex table in CDR, which covers many of the features of the helper. Some features in this is also using the GridHelper

Demo

This may seem like a long example, but it helps visualize that even though it's long, setting this up in a general non-helper way would be much, much longer and much harder to do and look at.
