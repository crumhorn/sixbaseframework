ContextMenuHelper

This AgGrid related helper, assists/simplifies setting up your context menus and at the same time hooking them in with any Toolbar implementation you may have.

Demo

It's easier to show some real world usage to understand how it works. Do note that this example uses actual images rather than web fonts. It's of course fine to use both or either.
