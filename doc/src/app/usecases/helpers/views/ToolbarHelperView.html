ToolbarHelper

This Toolbar related helper, assists with the creation of buttons and items for the Toolbar

The goal of the ToolbarHelper is to facilitate the setup of the Toolbar in a streamlined manner. It's easier to start with a "Demo". The example code is here a full setup of a rather complex table in CDR, which covers many of the features of the helper. This example uses images, but you can of course use web fonts or images both or separately.

Please take note of the GridToolbarHelper helper that deals with toolbar functions related to AgGrid. They are in a separate class so that dependencies are kept separate in case AgGrid is not in use.
